commonMark
directive in module hc.commonmark

Description

Compiles source test into HTML.

Usage

as element:
<common-mark
       common-mark="{expression}"
       [opts="{expression}"]>
</common-mark>
as attribute
<any common-mark="{expression}"
     [opts="{expression}"]>
   ...
</any>

Parameters

ParamTypeDetails
commonMarkexpression

The source text to be compiled. If blank uses content as the source.

opts
(optional)
expression

Hash of options that override defaults.
